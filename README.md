# git-sparse-check README
插件名称：git sparse check
 

## Features

用于git执行稀疏检出
如果monorepo项目想要拉取项目中某个应用文件夹，就可以使用稀疏检出的方式，避免手动执行多行代码，直接将该功能封成插件

## Requirements
 

## Extension Settings

在vscode窗口control+shift+p，输入git sparse命令并选中
根据提示分别选择：
保存的文件路径，
输入git仓库地址，
输入branch分支，
以及输入想要拉取的文件夹名称
